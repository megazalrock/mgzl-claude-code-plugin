## 概要

ユーザーが任意のタイミングで教訓蓄積を起動するための薄いラッパー。@knowledge-distiller サブエージェントを **fire-and-forget** で起動する。メインスレッドはエージェントの完了を待たず、起動指示を出したら即座に終了する。

バックグラウンドのエージェント完了通知はハーネスが自動でメインスレッドに届けるので、明示的な polling は不要。

## コンテキスト

- カレントセッションの直近のレビュー結果・指摘内容・修正履歴

## タスク

1. **抽出元（source）のまとめ**
   - カレントセッションの直近のレビュー結果・指摘・修正内容を要約して `source` としてまとめる
   - `source` が空になった場合は、「抽出元が見つからなかったためスキップしました」と伝えて終了する

2. **バックグラウンド起動**
   - `Agent` ツールを以下のパラメータで呼ぶ:
     - `subagent_type`: `knowledge-distiller`
     - `run_in_background`: `true`
     - `description`: `教訓蓄積`
     - `prompt`: 以下の構造で構成する

     ```
     source:
     <抽出元テキスト>

     context:
     <呼び出し元・対象機能・補足情報など>
     ```

3. **即時終了**
   - Agent 起動の応答を確認したら **結果を待たず**、ユーザーに「バックグラウンドで教訓蓄積を開始しました」と一行で伝えて終了する

## 注意事項

- `$MGZL_DIR` の検証はエージェント側で行うため、このスキルでは事前検証しない
- このスキル自体は教訓ファイルを直接読み書きしない（全てサブエージェントに委ねる）
- 他のスキル（`impl:create` / `impl:execute` / `review:diff` / `review:file`）は内部的にこのスキルを呼ばず、直接 `@knowledge-distiller` サブエージェントを起動する。このスキルはユーザー手動起動用のエントリポイント
